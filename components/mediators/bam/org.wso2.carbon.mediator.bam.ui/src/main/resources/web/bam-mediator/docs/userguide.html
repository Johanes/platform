BAM Mediator

BAM mediator can be used to intercept messages in WSO2 ESB for the purpose of WSO2 BAM.

Mediation Stats Configuration

Mediation stats should be enabled in the mediator if the user needs to get statistics about the messages passes through the mediator.

BAM Credential

The BAM mediator requires user credentials to login to the BAM receiver.

Transport

Transport related information required for the connection should be given.

Properties

Properties are a set of properties that can be extracted from the intercepted message by the mediator.
