HDFS - User Guide

WSO2 Carbon HDFS feature provides Apache HDFS based highly scalable second-generation distributed File System. HDFS supports Column Family based data model.

WSO2 Carbon HDFS helps user to create and manage NoSQL data stores with authentication and authorization. HDFS component uses Carbon User Manager as the user base.

HDFS feature comes with a management user interface. Management interface allows user to manage File System Objectss, Column Families and Columns.

Manage File System Objectss

File System Objects management allows user to list available key spaces. User can share, edit and delete File System Objectss using interface.

Add File System Objects

    Home-> Manage-> HDFS File System Objectss-> Add

List File System Objectss

    Home-> Manage-> HDFS File System Objectss-> List

Edit File System Objectss

User can alter File System Objects name, replication factor and replica replacement strategy.

Delete File System Objects

user can delete File System Objects after the confirmation.

Manage Column Family

User can get Column Family details by click on the File System Objects name

    Home-> Manage-> HDFS File System Objectss-> List-> File System Objects

In the Column Family summery page user can list all the Column Families related to the File System Objects. HDFS endpoint URL listed in summery page. User has to give HDFS endpoint URL (IP:PORT) and the username and password in the connection string.

Add Column Family

User can add more columns to the File System Objects by clicking Add a New Column Family link.

Edit Column Family

Column Family Edit allows user to modify Name, Replication Factor and Replica Placement Strategy of a Column Family.

Delete Column Family

User can delete Column Family after the confirmation.

Manage Column

User can list Column Family summery and manage Column in Column Family page.

List Column

    Home-> Manage-> HDFS File System Objectss-> List-> File System Objects-> Column Family

Add Column

Add new Column option allows user to add a Column to Column Family and set a Index Column and the validation class. In HDFS Index Column use to create a second index.

Edit Column

Edit Column allows user to modify Name, Index Name and Validation Class.

Delete Column

User can delete a Column after the confirmation.
