Manage Message Stores

The Message Stores acts as storage for ESB messages.Store mediator can be used To Store messages That are going through the ESB Mediation Path.
  1. In the navigator, under Manage/Service Bus, click Message Stores.The Manage Message Stores page appears.

Adding Message Store

  1. Click Add Message Store button Add Message Store page appears.

    2. Figure 1: Adding Message Store

  2. Enter the required details, and click Save. The Manage Message Stores page will display the added Message Store. [Note that required parameters will be depend on the Selected Message Store implementation]

    3. Figure 2: Message Store List

Message Store Operations

  1. When you click on an added Message Store list item.It will direct you to a page where Listed Messages appears. You can click view , delete messages from that page.
