Manage Message Processors

The Message Processors are set of processors that are intended to consume the ESB messages stored in ESB Message stores and do processing.

  1. In the navigator, under Manage/Service Bus, click Message Processors.The Manage Message Processors page appears.

Adding Message Processor

  1. Click Add Message Processor Add Message Processor page appears.

    2. Figure 1: Adding Message Processor

  2. Enter the required details, and click Save. The Manage Message Processors page will display the added Message Processors. [Note that required parameters will be depend on the Selected Message Processor implementation]

    3. Figure 2: Message Store List

Scheduled Message Forwarding Processor Operations

  1. Scheduled Message Processor Comes with a User interface to give user some manual control over message forwarding operations. To use the user interface It must be deactivated if its in the active state.Using the UI. User can view , delete and resend messages
