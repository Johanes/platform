How to setup project

1. Install Flex SDK

  • Download Flex SDK from adobe website. http://www.adobe.com/products/flex/flexdownloads/
  • Unzip the downloaded archive.(say c:\flex3)
  • Add c:\flex3\bin to your PATH system variable
  • Open up a command prompt and type mxmlc -version. You will be able to see the version of the flex sdk.

2. Get the latest WSO2 Data Sevice

  • Download the latest dataservice solution from http://wso2.org/projects/data-services-server/java
  • Unzip the downloaded archive.(say c:\wso2ds)

3. Get the latest license server sample

  • Get OSSLicenseService.dbs from https://wso2.org/repos/wso2/trunk/solutions/data-services/java/modules/samples/OSSLicenseManager/dbs
  • Replace the file c:\wso2ds\samples\OSSLicenseManager\dbs\OSSLicenseService.dbs with the new dbs file
  • Open a command prompt and navigate to c:\wso2ds\samples\OSSLicenseManager\
  • Type ant(You must have Apache Ant tool installed in your system)
  • Navigate to c:\wso2ds\bin and run wso2server.bat/wso2server.sh

4. Install/Configure Tomcat and Blazeds

  • You have two options for this
    1. Customize your existing Tomcat server
    2. Get the Blazeds configured Tomcat server
  • More information about how to customize your existing Tomcat Server can be found from http://opensource.adobe.com/wiki/display/c/Installation+Guide
  • You can download Turnkey which contins the Blazeds configured Tomcat instance from http://opensource.adobe.com/wiki/display/blazeds/Release+Builds

4. Diploy blazeds.war to Tomcat Webapp folder

  • Get the war file from https://wso2.org/repos/wso2/trunk/solutions/data-services/java/modules/samples/OSSLicenseManager/flexui/bin/
  • Diploy this in Tomcat
  • Now you will be able to access the License Manager from http://localhost:8400/blazeds/web/

4. Setting up InteliJ Idea Project

  • Follow the instructions from http://www.jetbrains.com/idea/docs/Creating_Flex_Applications_with_IntelliJ_IDEA.pdf
